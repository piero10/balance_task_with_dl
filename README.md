# balance_task_with_dl

Небольшой репозиторий с экспериментами/исследованиями по задачам на графах и DL (ноутбуки в `researches/`).

## Быстрый старт
- **Python**: 3.10+ (рекомендуется venv/conda)
- **Установка зависимостей**: если в проекте есть `requirements.txt` / `pyproject.toml` — установите зависимости стандартным способом.

## Структура
- **`researches/`**: Jupyter-ноутбуки и утилиты для экспериментов.

## Запуск
- Откройте нужный `.ipynb` в Jupyter/VS Code/Cursor и запускайте ячейки по порядку.

## Примечания
- Ноутбуки могут быть «рабочими черновиками»: результаты/пути к данным зависят от окружения и локальных файлов.
